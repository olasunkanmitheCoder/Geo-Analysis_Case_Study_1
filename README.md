# Land Cover, Flood and Soil Erosion Risk Mapping Using Remote Sensing and GIS

This project analyzes how land cover influences flood and soil erosion risks in Akure South, Nigeria using geospatial analysis and remote sensing.

Using multi-temporal Landsat imagery (1990–2025), GIS spatial modeling, and the RUSLE erosion model, the project identifies environmental risk zones and estimates population exposure to hydrological hazards.

## Workflow
The analysis demonstrates a full geospatial workflow including:

- Land Cover classification from satellite imagery

- Land cover change detection

- Flood susceptibility modeling using Multi-Criteria Decision Analysis (AHP)

- Soil erosion estimation using the RUSLE model

- Population exposure analysis using spatial overlay
